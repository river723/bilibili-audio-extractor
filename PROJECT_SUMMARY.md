# B站音频提取器 - 项目说明

## 🎯 项目概述

**B站音频提取器**是一个专业的Windows应用程序，用于从B站视频中提取高质量音频。项目采用统一安装方案，提供一键式依赖安装和打包功能。

## 📦 核心文件

| 文件 | 用途 | 说明 |
|------|------|------|
| `gui_extractor_simple.py` | 🎯 主程序 | 精简版B站音频提取器，支持大会员扫码登录 |
| `build_package.py` | 🛠 统一工具 | 集成依赖安装和打包功能 |
| `一键打包.bat` | 🚀 启动脚本 | 用户友好的安装和打包入口 |
| `verify_installation.py` | ✅ 验证工具 | 测试安装是否成功 |

## ✨ 核心功能

### 程序功能
- 🎵 **最高音质提取**：自动检测并下载最高音质音频
- 🔐 **大会员支持**：扫码登录获取更高音质权限  
- 📱 **智能二维码**：本地生成 + 外部API备选 + 文字链接备选
- ⚡ **一键操作**：输入URL → 点击提取 → 自动完成
- 📝 **详细日志**：实时显示处理进度

### 安装系统
- ✅ **统一安装**：一个脚本完成所有依赖安装
- ✅ **智能检测**：自动检查并安装缺失依赖
- ✅ **qrcode集成**：优先本地二维码生成
- ✅ **降级机制**：依赖缺失时自动使用备选方案

## 🚀 使用指南

### 方式1：完整打包（推荐）
```
1. 双击运行 "一键打包.bat"
2. 等待安装依赖和打包完成
3. 在 output/ 目录获取安装包
4. 分发给最终用户
```

### 方式2：仅安装依赖
```
运行: 一键打包.bat install
功能: 仅安装Python依赖，不打包
适用: 开发测试环境
```

### 方式3：验证安装
```
运行: python verify_installation.py
功能: 测试所有依赖是否正常工作
```

## 📋 安装包内容

构建完成后，`output/` 目录将包含：
- `Bilibili_Audio_Extractor_v2.1.0_Portable.zip` - 完整安装包

解压后包含：
- `B站音频提取器.exe` - 主程序
- `启动程序.bat` - 启动脚本
- `you-get/` - 视频下载库
- `使用说明.txt` - 详细文档
