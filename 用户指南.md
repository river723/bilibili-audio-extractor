# B站音频提取器 - 使用说明

## 🎯 应用简介

B站音频提取器是一个专门用于下载B站视频最高音质音频的工具。**主要程序 gui_extractor_simple.py** 支持B站大会员扫码登录，自动获取最高音质音频。

## ✨ 主要功能

### 🎯 核心程序：gui_extractor_simple.py

**gui_extractor_simple.py** 是本项目的主要程序，具有以下特色：

- **🎵 最高音质优先**：只下载输入URL的最高音质音频
- **🔐 B站大会员支持**：扫码登录获取更高音质权限  
- **⚡ 一键操作**：输入URL → 点击提取 → 自动完成所有步骤
- **📱 智能二维码**：本地生成 + 外部API备选 + 文字链接备选
- **📝 详细日志**：实时显示处理进度和状态信息

### 🎵 音频提取功能
- **最高音质**：自动选择最高质量的音频流
- **智能算法**：优先选择FLAC、WAV无损格式
- **多格式支持**：支持Hi-Res、Lossless、320kbps等高质量音频
- **自动转换**：输出为FLAC格式（失败时降级为MP3）

### 🛡️ 稳定性保障
- **本地二维码生成**：优先使用本地qrcode库（如果已安装）
- **外部API备选**：本地不可用时使用在线API
- **文字链接备选**：网络异常时显示可复制的登录链接
- **多层重试**：网络请求自动重试，提高成功率

## 📖 使用步骤

### 第一步：启动应用
```
双击运行 gui_extractor_simple.py
```

### 第二步：B站登录（可选但推荐）

#### 方式A：扫码登录
1. 点击界面上的 **"B站登录"** 按钮
2. 使用B站APP扫描弹出的二维码
3. 在APP中确认登录
4. 登录成功后即可下载更高质量音频

#### 方式B：链接登录（当二维码无法显示时）
1. 点击"B站登录"按钮
2. 如果二维码加载失败，会显示登录链接
3. 点击"复制链接"按钮
4. 将链接粘贴到浏览器中打开
5. 在浏览器中完成B站登录

### 第三步：提取最高音质音频

1. **输入URL**：在"B站视频链接"区域粘贴视频URL
   - 示例：`https://www.bilibili.com/video/BV1Hs4y1B7T2`

2. **选择目录**：点击"浏览"选择音频保存位置
   - 默认保存到：`用户目录/B站音频提取`

3. **开始提取**：点击"开始提取音频"按钮

4. **等待完成**：程序会自动：
   - 检测B站登录状态
   - 获取最高音质信息
   - 下载视频（使用大会员权限如果已登录）
   - 提取音频为FLAC格式
   - 保存到指定目录

## 🔧 高级功能

### 复制功能
- 当二维码无法显示时，可以复制登录链接
- 点击"复制链接"按钮，链接会自动复制到剪贴板
- 可以将链接粘贴到任何浏览器中完成登录

### 音质选择算法
程序使用智能评分算法选择最高音质：

| 音质类型 | 评分 | 优先级 |
|---------|------|--------|
| FLAC/WAV/AIFF | +1000 | 最高 |
| M4A/MP3 | +500 | 高 |
| 纯音频流 | +100 | 中 |
| Hi-Res | +50 | 中高 |
| Lossless | +40 | 中 |
| 320kbps | +30 | 中 |
| 128kbps | +10 | 低 |

### 多层备选方案

#### 二维码生成策略
1. **优先**：本地qrcode库（如果已安装）
2. **备选**：外部API（api.qrserver.com）
3. **最终**：文字链接（可复制到浏览器）

#### 音频格式策略
1. **优先**：FLAC无损格式
2. **备选**：MP3 320kbps高质量格式

## ❓ 常见问题

### 关于 gui_extractor_simple.py

**Q：为什么选择 gui_extractor_simple.py？**
**A：** gui_extractor_simple.py 是本项目的主要程序，专注于：
- 🎯 只下载最高音质音频（简化用户选择）
- 🔐 完整的B站大会员登录支持
- 📱 多层备选方案确保稳定性
- ⚡ 一键式操作体验

**Q：二维码无法显示？**
**A：** 可能原因：
1. 网络连接问题
2. 外部API暂时不可用
3. 防火墙阻止访问

**解决方案：**
- 使用文字链接方式登录（点击"复制链接"）
- 安装qrcode库使用本地生成
- 检查网络连接

**Q：如何获得最高音质？**
**A：** 
1. 使用B站大会员账号登录
2. 程序会自动选择最高音质
3. 优先下载FLAC无损格式

### Q：登录后还是没有高音质？
**A：** 可能原因：
1. 视频本身没有高音质版本
2. Cookie失效需要重新登录
3. 网络问题导致无法获取高音质

**解决方案：**
- 重新扫码登录
- 检查网络连接
- 尝试其他视频

### Q：音频提取失败怎么办？
**A：** 常见原因：
1. URL格式错误
2. 视频不存在或已删除
3. 网络连接问题
4. 依赖软件未安装

**解决方案：**
- 检查URL是否正确
- 确保you-get和ffmpeg已安装
- 检查网络连接
- 查看日志区域的错误信息
