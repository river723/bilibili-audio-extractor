## 📋 详细说明

### 安装程序使用方法

#### 便携版安装（推荐）

1. **解压安装包**
   ```
   将 ZIP 文件解压到任意目录
   例如: C:\BilibiliExtractor
   ```

2. **下载FFmpeg（必需）**
   - 访问: https://www.gyan.dev/ffmpeg/builds/
   - 下载: `ffmpeg-git-full.7z`
   - 解压并添加到系统PATH
   - 或将`ffmpeg.exe`放在程序目录

3. **运行程序**
   - 双击 `启动程序.bat`（中文版）
   - 或双击 `Launch.bat`（英文版）

### 音频质量选择

- **🎛️ 原始质量**：保持原始参数，音质最佳，文件较大
- **📻 CD音质**：标准CD音质，平衡音质和文件大小
- **🎵 MP3音质**：文件小，适合存储和分享

### 使用步骤

1. 双击桌面快捷方式或运行启动脚本
2. 在"视频URL"输入框粘贴B站链接
3. 选择输出目录（可选）
4. 选择音频质量
5. 选择是否保留视频文件
6. 点击"开始提取音频"
