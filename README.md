# B 站音频提取器

从 Bilibili 视频中提取高质量音频的一键工具，支持大会员扫码登录获取最高音质。

## ✨ 核心功能

- 🎵 **最高音质提取**：自动检测并下载 4K/1080P 高质量音频
- 🔐 **大会员支持**：扫码登录获取更高音质权限
- 📱 **智能二维码**：本地生成二维码，方便扫码登录
- ⚡ **一键操作**：输入视频 URL → 点击提取 → 完成

## 🚀 快速开始（3 步）

### 方式 1：完整打包（推荐分发）
1. 双击 `一键打包.bat`
2. 等待安装和打包完成（约 2-5 分钟）
3. 在 `output/` 目录获取安装包 `Bilibili_Audio_Extractor_v2.3.0_Portable.zip`
4. 解压后双击 `B 站音频提取器.exe` 开始使用

### 方式 2：仅安装依赖（开发测试）
```bash
一键打包.bat install
```

### 方式 3：验证安装
```bash
python verify_installation.py
```

## 📝 使用步骤

1. 打开 `B 站音频提取器.exe`
2. 输入 Bilibili 视频链接（如 `https://www.bilibili.com/video/BV1xxx`）
3. 点击「开始提取」
4. 选择音频格式和质量
5. 点击「提取」等待完成
6. 查看输出目录获取音频文件

## ❓ 常见问题

**Q: 扫码登录是必要的吗？**
A: 不是。不扫码可提取普通音质；扫码登录后可提取最高音质（4K/1080P）。

**Q: 安装失败怎么办？**
A: 检查网络连接，以管理员身份重新运行 `一键打包.bat`

**Q: 支持哪些视频链接？**
A: 支持 Bilibili 所有 BV 号或视频 ID 链接

**Q: 能否直接输出音频格式？**
A: 当前默认输出 MP3 格式，格式可在程序内设置
