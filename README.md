## 📋 详细说明

### 安装程序使用方法

#### 便携版安装（推荐）

1. **解压安装包**
   ```
   将 ZIP 文件解压到任意目录
   例如: C:\BilibiliExtractor
   ```

2. **下载FFmpeg（必需）**
   - 访问: https://www.gyan.dev/ffmpeg/builds/
   - 下载: `ffmpeg-git-full.7z`
   - 解压并添加到系统PATH
   - 或将`ffmpeg.exe`放在程序目录

3. **运行程序**
   - 双击 `启动程序.bat`（中文版）
   - 或双击 `Launch.bat`（英文版）

### 🎯 主要程序：gui_extractor_simple.py 功能特色

- **🎵 最高音质优先**：自动检测并下载最高音质音频
- **🔐 B站大会员支持**：扫码登录获取更高音质权限
- **📱 智能二维码**：本地生成 + 外部API备选 + 文字链接备选
- **⚡ 一键操作**：输入URL → 点击提取 → 自动完成所有步骤
- **📝 详细日志**：实时显示处理进度和状态信息

### 使用步骤

1. **启动主程序**
   - 运行 `gui_extractor_simple.py`

2. **B站登录（推荐）**
   - 点击"B站登录"按钮
   - 使用B站APP扫码登录
   - 获取大会员高音质权限

3. **输入视频URL**
   - 在"B站视频链接"输入框粘贴URL
   - 示例: `https://www.bilibili.com/video/BV1Hs4y1B7T2`

4. **选择输出目录**
   - 点击"浏览"选择保存位置
   - 默认: `用户目录/B站音频提取`

5. **开始提取**
   - 点击"开始提取音频"按钮
   - 程序自动完成下载和音频提取

6. **获取结果**
   - 输出FLAC无损格式（优先）
   - 或MP3高质量格式（备选）
   - 文件名: `视频标题_最高音质.flac`

## ❓ 常见问题

### 构建问题

**Q: 如何构建安装程序？**
A: 双击运行"一键打包.bat"，等待完成即可。

**Q: 需要安装Python吗？**
A: 用户不需要！安装包完全独立，包含所有依赖。

**Q: 支持哪些Windows版本？**
A: Windows 7/8/10/11，推荐64位系统。

**Q: 安装包有多大？**
A: 约35MB，包含所有必要文件。

**Q: 如何更新程序？**
A: 重新运行"一键打包.bat"生成新版本。

### 运行问题

**Q: 程序无法启动？**
A: 请确保已安装FFmpeg并添加到系统PATH。

**Q: FFmpeg not found？**
A: 下载FFmpeg并按照安装说明进行安装。

**Q: 下载失败？**
A: 检查网络连接，确保视频链接正确。

**Q: 输出文件在哪里？**
A: 在设置的输出目录中，文件名基于视频标题。

**Q: 如何获得最高音质？**
A: 使用B站大会员账号登录，程序会自动选择最高音质。

**Q: 二维码无法显示？**
A: 可使用文字链接方式登录，或安装qrcode库使用本地生成。

**Q: 如何卸载？**
A: 直接删除程序目录即可。
